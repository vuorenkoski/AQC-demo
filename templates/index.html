AQC demo

This demo runs some graph algorithms with the D-Wave leap heuristic local annealing simulator, and with the D-wave hybrid solver and quantum solvers in the cloud. The quantum solver can have long waiting-time to run if many qubits are needed. Demo can be run in three different D-Wave quantum solvers: Advantage_system4.1, Advantage_system5.4 and Advantage_system6.4. However, they are not necessarily all available.

The solver API token is needed if hybrid solver or quantum solvers are used. Solver API token can be aquired from the D-Wave leap dashboard: https://cloud.dwavesys.com/leap/
